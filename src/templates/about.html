Random-Lender

Random_Lender es una aplicación web que obtiene números aleatorios de la web numeroalazar.com.ar y los almacena en diferentes bases de datos:

  • MongoDB (local)

  • Beebotte (online)

  • MySQL (local) (obsoleto)

Random_lender nos permite realizar ciertas operaciones con los números almacenados:

  • Obtener número aleatorio

    Se nos muestra un numero aleatorio alamcenado en la base de datos.

  • Mostrar tabla

    Nos muestra una tabla con todos los números en la base de datos y sus respectivas fechas de obtención.

  • Umbral

    Indicado un número como umbral, se nos indicará cuál es el último número alamacenado en las bases de datos que lohaya superado tanto por arriba como por abajo.

  • Media

    Muestra la media de los números almacenados en las bases de datos.

  • Gráficas Remotas (Beebotte):

    Muestra gŕaficas de la web de Beebotte.

  • Gráficas Locales (Plotly):

    Muestra gráfica de los números aleatorios registrados. También permite mostrar gráfico de barras con las frecuencias de los números por intervalos. Se emplea la librería de plotly para javascript.

Random-Lender también nos avisará cuando se haya registrado un número que active el umbral del usuario, es decir, que sea mayor que el umbral del usuario (puede cambiarse en el apartado de cuenta, opción en la barra superior de la aplicación).

Umbrales de Usuario

Los umbrales funcionan de la siguiente manera:

  • Un umbral debe estar comprendido entre 0 y 100 para que sea válido.

  • La aplicación notificará al usuario cuando se registre un número mayor que el umbral que haya especificado.

    Ejemplo:
    Umbral Usuario: 50
    Número registrado: 40 → 40 < 50 → No avisa
    Número registrado: 60 → 60 > 50 → Avisa
