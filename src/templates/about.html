Random-Lender

Random_Lender es una aplicación web que obtiene números aleatorios de la web numeroalazar.com.ar y los almacena en diferentes bases de datos:

  • MongoDB (local)

  • Beebotte (online)

  • MySQL (local) (obsoleto)

Random_lender nos permite realizar ciertas operaciones con los números almacenados:

  • Umbral: Indicado un número como umbral, se nos indicará cuál es el último número alamacenado en las bases de datos que lohaya superado tanto por arriba como por abajo.

  • Media: Muestra la media de los números alamacenados en las bases de datos.

  • Grafo Beebotte: Redirige a la página de Beebotte y muestra grafos con los datos almacenados allí.

  • Grafo: Muestra gráfica de los números aleatorios registrados. También permite mostrar gráfico de barras con las frecuencias de los números por intervalos.

Random-Lender también nos avisará cuando se haya registrado un número que active el umbral del usuario (puede cambiarse en el apartado de cuenta).

Umbrales de Usuario

Los umbrales funcionan de la siguiente manera:

  • Un umbral debe estar comprendido entre -100 y 100 para que sea válido.

  • Si un umbral de usuario es mayor que 0, se notificará al usuario cuando se registre un número mayor que el umbral.

    Umbral Usuario: 50

    Número registrado: 40 => (40 < 50) => No avisa

    Número registrado: 60 => (60 > 50) => Avisa

  • Si un umbral de usuario es menor que 0, se notificará al usuario cuando se registre un número menor que el valor absoluto del umbral.

    Umbral Usuario: -50

    Número registrado: 40 => (40 < 50) => Avisa

    Número registrado: 60 => (60 > 50) => No avisa
